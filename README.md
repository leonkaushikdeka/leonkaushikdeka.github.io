# leonkaushikdeka.github.io

Public project showcase for Leon Kaushik Deka.

The site is designed as a direct-use hub for GitHub repositories. Each featured project should answer four questions quickly:

- What does it do?
- Who is it for?
- How can someone try, download, or deploy it?
- What is its current status and limitation?

## Publishing Model

- Static browser tools: GitHub Pages
- TypeScript apps: Vercel or Netlify
- Python demos: Hugging Face Spaces, Streamlit, Render, Railway, or Fly.io
- Flutter apps: GitHub Releases with APK files
- Rust CLI tools: GitHub Releases with versioned binaries
- Security-sensitive projects: documentation-first, authorized-use framing, and no unsafe public demo paths

## Local Preview

Open `index.html` in a browser. No build step is required.

## Deploy

GitHub Pages publishes this repository from the configured Pages branch.
